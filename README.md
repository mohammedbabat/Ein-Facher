# Ein-Facher Cloud – IT Support & Microsoft 365

Eine professionelle IT-Unternehmenswebseite für Cloud-Support und Microsoft 365 Dienstleistungen.

## 🌐 Live-Webseite
[ein-facher.cloud](https://ein-facher.cloud)

## 📁 Projektstruktur

```
Ein-Facher/
├── index.html      # Hauptseite
├── termin.html     # Terminbuchung
├── style.css       # Alle Styles
├── script.js       # JavaScript & Logik
└── CNAME           # GitHub Pages Domain
```

## ✨ Features

- Zweisprachig: Deutsch & Arabisch (DE / AR)
- Responsive Design (Mobile, Tablet, Desktop)
- Animierter Ladebildschirm
- Scroll-Reveal Animationen
- Interaktive Service-Tags mit Modal
- Kontaktformular via EmailJS
- Terminbuchungsseite

## 🛠️ Technologien

- HTML5, CSS3, JavaScript (Vanilla)
- [EmailJS](https://www.emailjs.com/) – Formular-E-Mails
- [Google Fonts](https://fonts.google.com/) – Outfit & Noto Kufi Arabic
- GitHub Pages – Hosting

## 🚀 Lokal starten

Kein Build-Tool nötig – einfach `index.html` im Browser öffnen.

```bash
git clone https://github.com/mohammedbabat/Ein-Facher.git
cd Ein-Facher
# index.html im Browser öffnen
```
